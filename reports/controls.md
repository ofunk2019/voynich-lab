**Purpose.** Learn what "nothing" looks like: each control embodies a trivial
explanation (it is Latin, it is a simple cipher, it is chance, etc.), and the T1
battery measures what it reproduces or fails to reproduce from the Voynich
signature. This is mandatory before interpretation (VOY-DOC-02).

**Equalization method.** The six corpora share the same layout skeleton:
3,601 lines, the same number of words per line, the same
paragraph starts, and therefore exactly 30,525 tokens each.
Size-sensitive measures (entropies, MATTR, hapax) are compared at strictly equal
size with the same estimator, which is the validity condition set in T1.

The five controls:

1. **Latin** - Clementine Vulgate (Genesis + Exodus), real medieval text: what a
   real language looks like in our battery.
2. **Enciphered Latin** - the same text under seeded monoalphabetic substitution:
   what a simple cipher changes (symbol identity) and does not change (everything
   else).
3. **Shuffled Voynich** - real manuscript words in random order: destroys
   sequential structure, keeps the lexicon.
4. **Gibberish** - glyphs drawn i.i.d. from real frequencies, with real word
   lengths: only the histogram remains.
5. **Self-citation** - each word is a mutated copy of an already written word
   (after Timm & Schinner): the "meaningless mechanical text" hypothesis embodied.

> **Pedagogy - z-score.** To compare measures with different units (bits,
> percentages, slopes), each value is expressed as "number of standard deviations
> above/below the group mean": that is the z-score. A control's distance to
> Voynichese on one metric is |z(control) - z(Voynich)|. The aggregate distance is
> the unweighted mean over the 15 metrics: the simplest
> possible choice, stated and documented.

> **VOY-DOC-07 - multiple comparisons.** 15 metrics x 5 controls =
> 75 comparisons. Isolated coincidences are expected; only coherent
> family-level patterns count.

## 4. What Each Control Teaches

Metric-by-metric reading of table 3, family by family:

- **Real Latin:** natural language has much higher h2, more varied word lengths,
  and fewer adjacent repetitions. Any Voynich/control gap here is a gap to "a real
  European language laid out the same way".
- **Enciphered Latin:** compare its column with clear Latin: distances are almost
  identical metric by metric. **This is T2's key lesson**: a substitution cipher
  moves no structural statistic (h1, h2, Zipf, lengths, repetitions). If
  Voynichese were a simple cipher of a European language, it would have that
  language's statistics. It does not.
- **Shuffled Voynich:** by construction, preserves everything that depends only on
  the lexicon (Zipf, lengths, hapax, MATTR, h1). What degrades under shuffling
  (adjacent repetitions, neighbour similarity, line effects, inter-word h2) is
  precisely the *sequential* part of the signature, the part a language or ordered
  procedure must explain.
- **Gibberish:** preserves only h1 (imposed histogram) and word lengths (the real
  length sequence is kept position by position, so "line effects" at 0.00 in table
  3 are construction artifacts, not chance succeeding). Everything else collapses:
  h2 rises toward h1 (without bigram structure, the previous glyph teaches
  nothing), Zipf slope flattens, vocabulary explodes (MATTR, hapax). Useful for
  seeing what "well-dressed randomness" does NOT do.
- **Self-citation:** with naive fixed parameters, inspect what it brings closer
  (repetitions, neighbour similarity, lengths) and what it still misses. T3 will
  search this family's parameter space.

**Interpretation guardrail (VOY-DOC-01/02).** This table ranks *families of
trivial explanations*, not truths: a low distance says "this family reproduces
these properties", never "the manuscript was produced this way".
